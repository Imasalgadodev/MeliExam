#### Versión de prueba

##### Pre-requisitos
- Java 8
- Un cliente http, preferentemente Postman

##### Pasos:
- Clonar repositorio
- Abrir con IntelliJ IDEA
- Ejecutar ( mvnw spring-boot:run )
- Ejecutar los datos de prueba
	- Importar la coleccion de postman ( datapostman\Meli.postman_collection.json )
